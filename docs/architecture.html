openmemory_rs System Architecture

Unified Rust-native MCP Cognitive Memory Engine — Integrating Graph, AST, Vector, Episodic & Shared Agent workspaces

1. Central Coordinator

  • • Coordinates all CRUD, AST, Vector, and log tables natively.
  • • Evaluates combined score ranking for semantic facts.
  • • Crawls directories to run static codebase parsing.
  • • Manages message routing for shared workspace databases.

2. SQLite Database Mappings

  • • graph_nodes / graph_edges: Explicit active relationships.
  • • semantic_metadata: Text fragments with model vector blobs.
  • • episodic_logs / reflection_memory: Attempt cycles & lessons.
  • • code_elements / code_calls: AST element & caller mappings.
  • • repository_evolution: Git Summaries & version tracking.
  • • shared_agent_memory: Key-Value cross-agent team board.

3. Advanced Cognitive Capabilities

* **Local Embeddings**: Safe ORT-backed inferences, zero cloud dependencies. * **Recency Decay**: Focuses attention by decaying old information. * **Agent Experience**: Remembers buggy setups, failed tools and reflections. * **Shared Memory Workspace**: Allows direct state synchronization for agents.
